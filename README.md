# Amazon Bestseller Data Analysis

This project analyzes Amazon bestselling books to explore whether the number of reviews or the price of a book affects user ratings. The analysis uses Python and the Pandas library to process and analyze the data.

## Questions

- Do number of reviews affect user ratings?
- Does price affect user ratings?

## Key Insights

- Average ratings remain relatively stable (around 4.5-4.7) across most review ranges, suggesting that popularity does not strongly influence future user ratings.
- Books with high review counts tend to be more variable. This suggests that exposure to a more diverse audience can result in more varied ratings.
- Price does not exhibit a strong correlation with user ratings, indicating that higher cost does not translate to higher perceived value.
- Mid-range priced books ($30-$50) show a dip while books price above $50 rise before dropping again at the $100 range.

## Conclusion

Overall, neither popularity nor price strongly determines user ratings. Instead, ratings remain consistently high across most categories, suggesting that other factors—such as content quality or genre may play a more influential role in determining user satisfaction. This analysis shows beginner authors that while popularity and price are important, they are not a major determinant of a book's success.
