# Amazon Bestseller Data Analysis

## Overview
Analyzed Amazon bestseller data to explore how popularity (number of reviews) and price impact user ratings.

## Tools Used
- Excel (Pivot Tables, Data Analysis, Dashboards)
- Data Visualization

## Key Questions
- Do more reviews affect ratings?
- Does price influence ratings?

## Key Insights
- Ratings remain stable (~4.5–4.7) for books with fewer than 40,000 reviews  
- Higher variability appears in highly reviewed books (70k–79k range dip)  
- No strong relationship between price and ratings  
- Books priced $40–49 had lower ratings, while $50–59 had the highest
